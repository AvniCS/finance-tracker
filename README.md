# 💸 Finance Tracker Web App

A simple finance tracker built using **Flask, Python, HTML, and CSS**.  
It helps users record their income, expenses, and see spending insights with graphs.

## 🚀 Features
- Add income & expenses
- Visualize spending with charts
- Email report generation
- User-friendly interface

## 🛠️ Tech Stack
- Python
- Flask
- HTML & CSS
