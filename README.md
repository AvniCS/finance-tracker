# 💸 Finance Tracker Web App

A simple finance tracker built using **Flask, Python, HTML, and CSS**.  
It helps users record their income, expenses, and see spending insights with graphs.

## 🚀 Features
- Add income & expenses
- Visualize spending with charts
- Email report generation
- User-friendly interface

## 🛠️ Tech Stack
- Python
- Flask
- HTML & CSS

# Install dependencies
- Make sure you have Python installed. Then run in bash:-
    - pip install -r requirements.txt

# Run the app in bash:-
- python app.py

# Open in your browser:-
- Go to http://127.0.0.1:5000 to start using the app.
